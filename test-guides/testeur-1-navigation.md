## Votre mission

Tester la **navigation générale** du site, l'affichage sur mobile et desktop, et vérifier que toutes les pages publiques fonctionnent.

---

## Tests à effectuer

### 1. Page d'accueil
- [ ] Ouvrir https://faitierehub.com
- [ ] La page s'affiche correctement (pas d'écran blanc)
- [ ] La vidéo ou l'image de démonstration est visible
- [ ] Les boutons "Essai gratuit" et "Voir les comptes d'exploitation" fonctionnent

### 2. Menu de navigation (desktop)
- [ ] Cliquer sur "Produit" → page s'affiche
- [ ] Cliquer sur "Sécurité" → page s'affiche
- [ ] Cliquer sur "Entreprise" → page s'affiche
- [ ] Cliquer sur "À propos" → page s'affiche (photo du fondateur visible)
- [ ] Cliquer sur "Contact" → page s'affiche (numéro +228 92 54 88 38)

### 3. Menu mobile (sur téléphone)
- [ ] Ouvrir le site sur téléphone
- [ ] Le menu hamburger (☰) apparaît
- [ ] Cliquer dessus → les liens s'affichent
- [ ] Chaque lien fonctionne

### 4. Comptes d'exploitation (page publique)
- [ ] Aller sur la page "Comptes d'exploitation" depuis le menu
- [ ] La page affiche les fiches disponibles (ou "Aucune fiche disponible")
- [ ] Les filtres (culture, préfecture) fonctionnent
- [ ] Le bouton "Accès membre" ouvre un dialogue

### 5. Pages légales (footer)
- [ ] Cliquer sur "Confidentialité" dans le footer → page s'affiche
- [ ] Cliquer sur "Conditions" → page s'affiche
- [ ] Cliquer sur "Cookies" → page s'affiche

### 6. Connexion / Déconnexion
- [ ] Cliquer sur "Se connecter"
- [ ] Entrer les identifiants ci-dessus
- [ ] Vérifier que ça redirige (même si accès limité pour invité)
- [ ] Se déconnecter → retour à la page de connexion
